# 美术规范 · deskPet

> 这是所有美术资源（无论 AI 生成还是外包）必须遵守的统一规范。**先把柴犬 idle 第一帧做出来锁样板，再发散其他资源**。

## 1. 风格关键词（用于 AI 绘图 prompt）

正向词（必带）：
- `手绘扁平插画风`、`温暖配色`、`暖橙暖黄主色`、`粗白描边`、`无渐变`、`无阴影`、`圆润柔和`
- `Q版`、`可爱`、`治愈`、`儿童绘本风`
- 英文备用：`flat illustration, warm palette, thick white outline, no gradient, no shadow, soft rounded shapes, kawaii, children's book style`

反向词（必排除）：
- `写实`、`照片`、`3D`、`金属反光`、`复杂阴影`、`恐怖`、`暗黑`、`成人`
- 英文：`photorealistic, 3D render, complex shadows, dark, horror, scary, adult content`

## 2. 画布与帧规格

| 项目 | 规格 |
|---|---|
| 画布尺寸 | **256 × 256 像素**，PNG 透明背景 |
| 安全区 | 中心 200×200 之内，外圈 28px 留作动作偏移 |
| 帧率 | 8–12 fps（待机 10 fps 即可） |
| idle 帧数 | 6–8 帧（呼吸 + 眨眼 + 摇尾轻微变化）|
| eat 帧数 | 8–12 帧（低头吃 → 抬头嚼 → 满足）|
| play 帧数 | 8–12 帧（跳跃或摇尾打转）|
| 升级动画 | 6 帧通用（星星/特效，不含狗狗本体）|
| 情绪小表情 | 静图 1 帧/种，64×64 像素，叠在狗狗头侧 |
| 旅行风景 | 1200 × 800 像素 JPG，无透明 |
| 明信片底板 | 1200 × 800 像素 PNG，带相框/邮票位 |
| 气泡框 | 9-slice 可拉伸 PNG，至少 128×64 |
| 软件图标 | 256×256 + 多分辨率 `.ico`，狗狗头像版 |

## 3. 文件命名约定

```
assets/breeds/{breed}/{action}/frame_{NNN}.png    # 三位数序号，001 起
assets/postcards/scenes/{destination}_{NN}.jpg    # 目的地缩写 + 序号
assets/postcards/templates/template_{NN}.png
assets/ui/bubble_{NN}.png
```

`{breed}` 取值：`shiba` `corgi` `golden` `husky` `teddy` `zhongtian`
`{action}` 取值：`idle` `eat` `play`

## 5. AI 绘图执行步骤

1. **样板锁定**（仅做一次）：用即梦/文心，输入正向词 + 反向词 + 「柴犬 站立 正面 idle」，生成 8–12 张候选，挑 1 张满意的。
2. **记录这次的 prompt 完整原文与 seed/参数**，写到本文件「6. 已锁定样板」段下面。
3. **同品种其他帧**：复用同一 prompt + seed，只改动作描述（如「张嘴吃东西」），让画风保持一致。
4. **其他品种**：把正向词里的「柴犬」改成对应品种，反向词不变。
5. **后期对齐**：每帧导出后在 GIMP/Photoshop 里居中到 256×256 透明画布，避免位移抖动。
